## project-1-Wahid_Jamaalii_As_Shidqi

### project-1-Wahid_Jamaalii_As_Shidqi

Ini adalah project 1 yang berisi tentang web biodata sederhana dari html, dan css

Di dalama **biodata** ini ada _data-data_ mengenai saya :

1. Nama
2. Tempat/Tanggal lahir
3. Agama
4. Jenis Kelamin
5. Golongan darah
6. No.Handphone
7. Pendidikan Terakhir

Di bagian **Navbar** ada pilihan :
* Home
* Profile
